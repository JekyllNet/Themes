# Landing Page Themes / 落地页主题

This directory contains **landing page themes** for JekyllNet.  
此目录包含 JekyllNet 的**落地页主题**。

Landing page themes are designed for single-page marketing or promotional sites.  
落地页主题适用于单页营销或推广站点。

---

## Submitting a Theme / 提交主题

Place your theme in a subdirectory: `themes/landing/<your-theme-name>/`  
将主题放置在子目录中：`themes/landing/<你的主题名称>/`

Each theme directory must contain a `README.md`. See the [Contributing Guide](../../CONTRIBUTING.md) for requirements.  
每个主题目录必须包含 `README.md`。要求详见[贡献指南](../../CONTRIBUTING.md)。
