# Documentation Themes / 文档主题

This directory contains **documentation themes** for JekyllNet.  
此目录包含 JekyllNet 的**文档主题**。

Documentation themes are designed for technical documentation sites.  
文档主题适用于技术文档站点。

---

## Submitting a Theme / 提交主题

Place your theme in a subdirectory: `themes/documentation/<your-theme-name>/`  
将主题放置在子目录中：`themes/documentation/<你的主题名称>/`

Each theme directory must contain a `README.md`. See the [Contributing Guide](../../CONTRIBUTING.md) for requirements.  
每个主题目录必须包含 `README.md`。要求详见[贡献指南](../../CONTRIBUTING.md)。
