# Portfolio Themes / 作品集主题

This directory contains **portfolio themes** for JekyllNet.  
此目录包含 JekyllNet 的**作品集主题**。

Portfolio themes are designed for showcasing personal or professional work.  
作品集主题适用于展示个人或专业作品。

---

## Submitting a Theme / 提交主题

Place your theme in a subdirectory: `themes/portfolio/<your-theme-name>/`  
将主题放置在子目录中：`themes/portfolio/<你的主题名称>/`

Each theme directory must contain a `README.md`. See the [Contributing Guide](../../CONTRIBUTING.md) for requirements.  
每个主题目录必须包含 `README.md`。要求详见[贡献指南](../../CONTRIBUTING.md)。
