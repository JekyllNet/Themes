# Blog Themes / 博客主题

This directory contains **blog themes** for JekyllNet.  
此目录包含 JekyllNet 的**博客主题**。

Blog themes are designed for personal or professional blogging sites.  
博客主题适用于个人或专业博客站点。

---

## Submitting a Theme / 提交主题

Place your theme in a subdirectory: `themes/blog/<your-theme-name>/`  
将主题放置在子目录中：`themes/blog/<你的主题名称>/`

Each theme directory must contain a `README.md`. See the [Contributing Guide](../../CONTRIBUTING.md) for requirements.  
每个主题目录必须包含 `README.md`。要求详见[贡献指南](../../CONTRIBUTING.md)。
