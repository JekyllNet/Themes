# Business Themes / 商业主题

This directory contains **business themes** for JekyllNet.  
此目录包含 JekyllNet 的**商业主题**。

Business themes are designed for corporate websites and professional services.  
商业主题适用于企业网站和专业服务站点。

---

## Submitting a Theme / 提交主题

Place your theme in a subdirectory: `themes/business/<your-theme-name>/`  
将主题放置在子目录中：`themes/business/<你的主题名称>/`

Each theme directory must contain a `README.md`. See the [Contributing Guide](../../CONTRIBUTING.md) for requirements.  
每个主题目录必须包含 `README.md`。要求详见[贡献指南](../../CONTRIBUTING.md)。
