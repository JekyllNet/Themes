# Minimal Themes / 极简主题

This directory contains **minimal themes** for JekyllNet.  
此目录包含 JekyllNet 的**极简主题**。

Minimal themes emphasize simplicity, clean design, and fast load times.  
极简主题强调简洁设计和快速加载。

---

## Submitting a Theme / 提交主题

Place your theme in a subdirectory: `themes/minimal/<your-theme-name>/`  
将主题放置在子目录中：`themes/minimal/<你的主题名称>/`

Each theme directory must contain a `README.md`. See the [Contributing Guide](../../CONTRIBUTING.md) for requirements.  
每个主题目录必须包含 `README.md`。要求详见[贡献指南](../../CONTRIBUTING.md)。
